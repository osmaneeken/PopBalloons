# PopBalloons

 You have 60 seconds. Let's see how many balloons you can pop.

# Game Information

-Orange balloon = +1 point

-Green  balloon = +5 point

-Black  balloon = Game Over

-Game speed increases every 10 seconds

-Game time 60 seconds
